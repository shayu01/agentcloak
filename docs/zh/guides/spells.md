# Spell

Spell 将常见站点操作封装为一行命令。可以用声明式管道 DSL 或异步 Python 函数编写，支持五种策略级别：PUBLIC、COOKIE、HEADER、INTERCEPT 和 UI。

本指南将涵盖 spell 系统架构、使用 `@spell` 装饰器编写自定义 spell、管道 DSL 语法、内置和用户目录的自动发现，以及用于生成 spell 模板的 scaffold 命令。

详细内容将在后续更新中补充。`spell list`、`spell run`、`spell info` 和 `spell scaffold` 命令参见 [CLI 参考](../reference/cli.md#捕获与-spell)。
