# Network capture and API analysis

agentcloak can record network traffic, export it as HAR 1.2, and auto-detect API patterns. Captured patterns can be used to generate spells -- reusable automation commands that call APIs directly.

This guide will cover the capture workflow (start/stop/export/analyze), HAR import/export, pattern analysis (endpoint clustering, auth detection, schema inference), and the spell generation pipeline.

Detailed content will be added in a future update. See the [CLI reference](../reference/cli.md#capture-and-spells) for available capture commands.
