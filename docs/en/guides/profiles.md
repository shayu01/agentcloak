# Browser profiles

Profiles let you save and restore browser sessions across runs. A profile stores cookies, localStorage, and other browser state in a named directory under `~/.agentcloak/profiles/`.

This guide will cover creating profiles from scratch or from a running session, launching with saved profiles, managing multiple profiles for different accounts or sites, and cookie export/import between profiles and backends.

Detailed content will be added in a future update. See the [CLI reference](../reference/cli.md#profile-management) for profile commands.
