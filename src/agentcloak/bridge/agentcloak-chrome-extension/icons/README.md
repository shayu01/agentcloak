# Extension Icons

These are placeholder 1x1 transparent PNGs. Replace with the real
agentcloak brand mark in three sizes:

| File      | Size (px) | Use                                                              |
|-----------|-----------|------------------------------------------------------------------|
| `16.png`  | 16x16     | Toolbar button + favicon on `chrome://extensions`                |
| `48.png`  | 48x48     | Extension management tile                                        |
| `128.png` | 128x128   | Web Store listing + install dialog (also used for `default_icon`)|

Chrome will still load the extension with placeholder icons present — the
buttons will just look empty. Swap them out before publishing.
